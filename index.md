# 伊藤 俊英 (Toshihide Itoh, Ph.D.)

- 医用画像工学の研究に30年以上従事しています。

- 専門はX線CT装置の研究開発です。特に以下のような計測プロセスを得意としています。
  - 物理シミュレーション
  - 検出器物性物理解析
  - 計測データプロセッシング開発
  - データ補正や画像再構成のアルゴリズム開発
  - スペクトラルイメージングアルゴリズムの開発
  - フォトンカウンティングCTの検出器物性、データ計測、補正アルゴリズムなどの開発

- 2010年からフォトンカウンティングCTの研究開発に従事しています。
